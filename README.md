### 项目结构
```
  .src
  |---models mobx的store
  |---components 组件
  |---pages 页面
  |---route 路由配置
  |---services 服务请求
  |---stores mobx的store
  |---utils 公共方法
  |---app.tsx 主页面路由
  |---index.tsx 入口文件
  |---pages.tsx 登录和主页面路由
```

### 运行
- 本地运行:

1.yarn install 或 npm install

2.yarn start 或 npm run start

- 打包

yarn build 或 npm run build

### 浏览器打开
http://localhost:3000/
